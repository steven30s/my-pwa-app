# 智能财务助手

个人财务管理应用，支持收支记录、统计分析和报表生成。

## 功能特性

- 收支记录管理
- 财务数据可视化
- 交易记录查询
- 分类统计报表

## 安装到手机主屏幕

本应用支持PWA(渐进式Web应用)功能，可以像原生应用一样安装到手机主屏幕。

### Android设备安装步骤：
1. 使用Chrome浏览器打开应用
2. 点击浏览器菜单(右上角三个点)
3. 选择"添加到主屏幕"
4. 确认安装

## 开发环境设置

1. 克隆仓库
```bash
git clone <repository-url>
```

2. 安装依赖
```bash
pnpm install
```

3. 启动开发服务器
```bash
pnpm dev
```

4. 构建生产版本
```bash
pnpm build
```

## 部署

构建后的文件位于`dist/static`目录，可以部署到任何静态文件服务器。

推荐部署平台：
- Vercel
- Netlify
- GitHub Pages
